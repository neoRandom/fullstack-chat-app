# Full-stack Chat App (Chatty)

### Made with:

- Front-end:
    - Vite
    - React
    - Lucid React
    - Hot Toaster
    - TypeScript
    - Zustand
    - TailwindCSS
    - DaisyUI
    - Axios

- Back-end:
    - Node
    - Express
    - Multer
    - Socket.io
    - Mongoose
    - MongoDB
